Transcribe audio without sending it to the cloud

Private, local, real-time transcription for macOS

Why inkoscribe?

100% Private Audio Transcription

Your voice data never leaves your Mac. No cloud uploads, no servers, no privacy risks.

Real-time Speech-to-Text

Instant, word-by-word transcription as you speak. Perfect for live meetings and calls.

Easy Installation - No Setup Required

Ready to use in seconds. No accounts, no configuration, no hassle.

Installation

⚠️ Requires macOS 13+ (M1, M2, M3+)

$ brew tap uohzxela/inkoscribe
$ brew install inkoscribe
$ inkoscribe -s sys # Transcribe system audio

Why I Built This

I'm Alex, a hearing-impaired software engineer who needed a local-first, real-time transcription tool for daily use. I built inkoscribe to transcribe meetings, interviews, and calls entirely on my machine — no risk of sending sensitive audio to the cloud. It's my daily driver, and I hope it can be yours too.
