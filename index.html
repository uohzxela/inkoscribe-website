inkoscribe

Transcribe audio without sending it to the cloud

Private, local, real-time transcription for macOS

Why inkoscribe?

100% Private

Your voice data never leaves your Mac. No cloud uploads, no servers, no privacy risks.

Real-time

Instant, word-by-word transcription as you speak. Perfect for live meetings and calls.

No Setup Required

Install with three commands and start transcribing immediately. No accounts, no configuration.

Install

Install and run in seconds:

$ brew tap uohzxela/inkoscribe
$ brew install inkoscribe
$ inkoscribe -s sys # Transcribe system audio
